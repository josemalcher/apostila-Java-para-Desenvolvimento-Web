
# Apostila do Curso FJ-21 - Java para Desenvolvimento Web

https://www.caelum.com.br/apostila-java-web/

O mercado Web é o mais forte no Java e o com mais oportunidades. Aprenda a criar uma aplicação Web completa com tecnologias importantes como Servlets, JSP, Spring e Hibernate, além de técnicas como Design Patterns, Ajax e MVC. Este material gratuito é o que usamos no curso de Java Web da Caelum e esperamos que seja útil no seu aprendizado. Não deixe também de compartilhar essa apostila com seus amigos.
